# Test: End Continuation

## Purpose
Test completion when cursor is at the end of a document, continuing an incomplete sentence/section.

## Expected Behavior
- Should continue the incomplete sentence naturally
- Should maintain the established tone and style
- Should provide practical next steps as implied by the section heading
- Content should be coherent with everything that came before

## Cursor Position
End of document, after "you'll need"

## Notes
This tests the most common use case - continuing to write at the end of a document.
